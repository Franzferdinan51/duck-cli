# Messaging Workflow

Use this for any recipient:
1. Open Messages.
2. Locate the conversation by contact/search.
3. Read the latest incoming message if replying.
4. Compose the reply or new text.
5. Send it.
6. Confirm completion.

Fallbacks:
- If UI search fails, use screenshot/vision to locate the conversation list.
- If the contact is ambiguous, ask the user which recipient.
